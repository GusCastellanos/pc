Dell PC Troubleshooting

These are steps you can take to help troublehoot reboot issues on a PC.

# Action Item What it Does Instructions and Resources
1 Clean PC vents and fans PCs will stop and restart if they get too hot. Cleaning your PC case and vents prevents the machine from overheating. Shut down and unplug the pc. Wipe all sides of the case and any vents with a microfiber or other lint free cloth. Look inside the case to check for significant dust build up. If you see a lot of dust build up (like lint in a lint trap) you will need to open up the PC and carefully remove the build-up without damaging sensitive electronic parts. After cleaning, check the location of the computer to be sure it has at least 3-4 inches of space on all sides to allow for proper air flow then plug in the machine and turn it on.
2 Disable Automatic Restarts By default when your PC hits a fatal error it will restart in the background and ask you to login again, so you don't see what error caused the restart. By disabling automatic restarts you force the PC to show you the error it encountered. Changing this setting on the pc means that when it hits any fatal error you will see a message and then have to manually restart to continue working. PDF file: Disable Automatic Restarts on a Dell PC.pdf
3 Run Dell Preboot Diagnostics and Hardware Tests Dell PCs have a built-in diagnostic screen that can check internal components for problems, such as Motherboards, Video Cards, and Memory. The diagnostic tests can tell you if all of those components are working properly or not. If a component fails the diagnostic test, it means it needs to be replaced. If all of the hardware components pass the diagnostic tests, the problem is likely caused by a piece of software. Dell article on running diagnostics
4 Disable Startup Apps and Background Programs Startup Apps and Background programs normally run quietly in the background to speed up large program load times by pre-loading some components but if they hit a fatal error they can cause unexpected system reboots. To Disable Startup Apps: Press Ctrl + Shift + Esc to open the Task Manager. Go to the Startup tab (or the Startup Apps icon on the left in Windows 11), right-click unnecessary programs and select Disable. Restart your PC for the new settings to take effect. Almost all startup apps can be disabled safely. If you notice a problem after disabling a startup app you can return to this same screen and select "Enable" to re-enable the app and restart again.

To Disable Background Permissions: Go to Start > Settings > Privacy > Background apps (or Privacy & Security in Windows 11), and toggle Let apps run in the background to Off. Restart your PC for the new settings to take effect.
